# خطوط Yemen Market

## الخطوط الموصى بها:
1. **Cairo**: خط عربي حديث وجميل
2. **Tajawal**: خط عربي أنيق

## كيفية إضافة الخطوط:
1. قم بتحميل الخطوط من:
   - https://fonts.google.com/specimen/Cairo
   - https://fonts.google.com/specimen/Tajawal

2. أنزلها كـ .ttf files
3. ضعها في هذا المجلد
4. أضفها في pubspec.yaml

## ترخيص الخطوط:
جميع الخطوط من Google Fonts مجانية للاستخدام التجاري.
